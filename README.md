# my-projet
Student Performance - Alcohol Consumption Prediction

Description

Ce projet vise à analyser les facteurs influençant la consommation d'alcool des étudiants et à prédire leur niveau de consommation à l'aide de techniques de Machine Learning.

Dataset

Student Performance Dataset (UCI Machine Learning Repository).

Outils Utilisés

Python

Pandas

Matplotlib

Scikit-learn

Objectif

Construire un modèle capable de prédire la consommation d'alcool des étudiants à partir de données académiques et sociales.
